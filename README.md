# Gordon's Bank (gordons-bank)

Projeto scaffold Angular gerado automaticamente.

## Como usar

1. Instale dependências:
   ```bash
   npm install
   ```
2. Rodar em modo dev:
   ```bash
   npm start
   # ou
   ng serve --open
   ```
3. Observação: este scaffold é simples — para um projeto real, rode `ng new gordons-bank` com Angular CLI e substitua/compare os arquivos.
